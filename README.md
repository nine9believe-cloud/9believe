# 9believe — เว็บแอปสั่งเครื่องดื่ม ☕

เว็บแอปสำหรับร้าน 9believe Home Cafe สร้างตาม design จาก Claude Design แบบเป๊ะทุกหน้าจอ
ใช้งานบนมือถือเป็นหลัก มีทั้ง **ฝั่งลูกค้า** (สั่งเครื่องดื่ม) และ **ฝั่งร้าน** (จัดการออเดอร์)

## มีอะไรบ้าง

**ฝั่งลูกค้า** (เปิดที่ `/`)
- หน้าแรก: ทักทายตามช่วงเวลา, ป้ายสถานะร้านเปิด/ปิด, หมวดเมนู, การ์ดเมนูพร้อมรูปจริง
- กดเมนู → Bottom Sheet เลือกความหวาน / นมโอ๊ต +฿10 / จำนวน / หมายเหตุ
- ตะกร้า → สรุปคำสั่งซื้อ (กรอกชื่อ เบอร์ บ้านเลขที่) → ชำระเงินด้วย QR พร้อมเพย์
  (มีนับถอยหลัง 5 นาที + ปุ่มดาวน์โหลด QR) → แนบสลิปจริง → ติดตามสถานะแบบสด
- พอร้านกด "จัดส่งสำเร็จ" ลูกค้าให้ดาว + เขียนรีวิว + แนบรูปได้

**ฝั่งร้าน** (เปิดที่ `/admin` — ใส่รหัส PIN)
- เปิด/ปิดร้านชั่วคราวได้ด้วยสวิตช์เดียว
- เห็นออเดอร์เข้าใหม่ พร้อม **รูปสลิปที่ลูกค้าแนบ** และยอดที่ต้องได้รับ
- กดปุ่มเดียวเพื่อเลื่อนสถานะ: ยืนยันการชำระ → กำลังชง → เตรียมจัดส่ง → กำลังจัดส่ง → จัดส่งสำเร็จ
  (ลูกค้าเห็นสถานะเปลี่ยนบนมือถือทันที)
- อ่านรีวิวจากลูกค้าทั้งหมด

ข้อมูลทุกอย่าง (ออเดอร์ สลิป รีวิว) เก็บจริงในฐานข้อมูล SQLite — ไฟล์เดียวในโฟลเดอร์ `data/`
ไม่ต้องติดตั้งฐานข้อมูลแยก

## วิธีรันบนเครื่องตัวเอง

1. ติดตั้ง [Node.js](https://nodejs.org) (เวอร์ชัน 20 ขึ้นไป) — ติดตั้งครั้งเดียว
2. เปิด Terminal เข้ามาที่โฟลเดอร์นี้ (`webapp`) แล้วพิมพ์ทีละบรรทัด:

```bash
npm install
npm run build
npm start
```

3. เปิดเบราว์เซอร์ไปที่ **http://localhost:3000** (ฝั่งลูกค้า)
   และ **http://localhost:3000/admin** (ฝั่งร้าน)

ถ้าจะแก้โค้ดแล้วดูผลทันที ใช้ `npm run dev` แทนสองคำสั่งหลังได้

## การตั้งค่า (ไม่บังคับ)

สร้างไฟล์ชื่อ `.env.local` ในโฟลเดอร์นี้ แล้วใส่ค่าที่อยากเปลี่ยน:

```bash
# รหัส PIN เข้าหน้าร้าน (ค่าเริ่มต้นคือ 9999 — ควรเปลี่ยนก่อนใช้จริง!)
ADMIN_PIN=1234

# เวลาเปิด–ปิดร้าน (ชั่วโมง เวลาไทย) ค่าเริ่มต้น 9 ถึง 17
SHOP_OPEN_HOUR=9
SHOP_CLOSE_HOUR=17
```

หมายเหตุ: ข้อความบนหน้าจอเขียนไว้ว่า "9:00–17:00" ตาม design
ถ้าเปลี่ยนเวลาทำการ ให้แก้ข้อความในโค้ดด้วย (ค้นหาคำว่า `9:00–17:00` ในโฟลเดอร์ `src/`)

## อยากแก้เมนู / รูป / QR

- **เมนูและราคา** — แก้ไฟล์ `src/lib/data.ts` (รายการ `MENU` อ่านง่าย แก้ชื่อ/ราคา/คำอธิบายได้เลย)
- **รูปเครื่องดื่ม** — วางรูปทับไฟล์ใน `public/images/menu/` (ตั้งชื่อไฟล์ตาม id ของเมนู เช่น `matcha-latte.webp`)
- **QR พร้อมเพย์** — วางรูปทับ `public/images/payment-qr.jpg`
- **โลโก้** — `public/images/9believe-logo-horizontal.png` และ `9believe-logo-mark.png`

## เอาขึ้นออนไลน์ยังไง

แอปนี้เก็บข้อมูลเป็นไฟล์ในโฟลเดอร์ `data/` จึงต้องใช้โฮสต์ที่มี **ดิสก์ถาวร** เช่น
[Railway](https://railway.app), [Render](https://render.com) (เลือกแบบมี Persistent Disk)
หรือเครื่อง VPS ทั่วไป — ตั้งค่า Build command เป็น `npm run build` และ Start command เป็น `npm start`
(⚠️ Vercel แบบฟรีไม่เหมาะ เพราะไฟล์ที่อัปโหลดจะหายเมื่อรีสตาร์ต)

อย่าลืมตั้ง `ADMIN_PIN` เป็นรหัสของตัวเองตอนขึ้นออนไลน์

## โครงสร้างโปรเจกต์ (สำหรับนักพัฒนา)

```
webapp/
├── src/lib/          ข้อมูลเมนู, ฐานข้อมูล, ตัวช่วยต่างๆ
├── src/components/   ชิ้นส่วน UI ตาม design system (ปุ่ม ฟอร์ม ไอคอน ฯลฯ)
├── src/app/          หน้าจอทั้งหมด + API
│   ├── (customer)/   หน้าฝั่งลูกค้า (แรก เมนู ตะกร้า ชำระเงิน ติดตาม รีวิว)
│   ├── admin/        หน้าฝั่งร้าน
│   └── api/          REST API (ออเดอร์ สลิป สถานะ รีวิว)
├── public/           รูปเมนู โลโก้ QR ฟอนต์ LINE Seed Sans TH
└── data/             ฐานข้อมูล + รูปที่ลูกค้าอัปโหลด (สร้างอัตโนมัติ ห้าม commit)
```
